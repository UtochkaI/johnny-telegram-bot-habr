# Johnny Article Origin Note

Status: user-confirmed origin context.

The Johnny Telegram Bot project did not start as a planned product. It started as an attempt to reuse an already rented VPS after it became unsuitable for VPN usage.

Provider:
- 62Yun
- URL: https://62yun.ru/

User-reported problems:
- VPS instability / outages
- frequent DDoS-related disruption
- provider maintenance windows
- support ignored or poorly handled requests
- provider website was not always available
- the site was unavailable again shortly before this note was written

Financial context:
- The server was originally rented for one month.
- Around 1200 RUB remained on the account.
- The balance could not be withdrawn.
- The user continued renting/using the VPS instead of abandoning the paid resource.

Initial prompt context:
- The user asked ChatGPT, approximately: "Come up with something non-obvious and funny."
- The first stage was not a finished Telegram bot idea and not the Johnny character.
- It began as a simple experiment: what can be built on this otherwise annoying VPS?

Article framing:
