# Publication safety note

This repository is intentionally limited.

The production Johnny bot is not published here because it is planned for live use.

Excluded by design:

- system prompt
- runtime prompt
- character bio prompt
- production code
- .env
- model/provider parameters
- chat IDs and sender IDs
- raw Telegram logs
- SQLite memory/state
- side-effect implementation details
- full patch marker index
- full source digest
- private user messages
